wiki的形成思路

首先，wiki的 Hexo 模板与整个blog的模板是不同的。所以面临的第一个问题是怎么将 Hexo blog 部署到 git repo 的子目录中。我使用的方法是使用脚本本地复制到总 blog 的根目录下。

第二个问题是：执行 Windows 批处理命令后步骤中的命令不运行。

这是因为一个步骤中的所有命令都放在将由服务器运行的 Windows 批处理文件中。如果该步骤命令包含对批处理文件的引用，那么在该批处理文件运行后该步骤将退出。在批处理文件引用后的步骤命令不会运行。暂时使用call xx.bat来解决。

第一个bat：找到主博客，生成静态文件。

第二个bat：找到wiki的博客目录，生成静态目录。

第三个bat：拷贝wiki博客的所有生成文件到主博客的wiki目录下

第四个命名为blog.bat：为了方便执行，前三个bat放在D盘，最后一个放在C:\widows目录下，这样以后直接打开CMD，输入blog就能直接上传到博客了。
